## Opis zadania
Moim zadaniem bylo stworzenie repozytorium w githubie zawierajace pliki z roznymi rozszerzeniami. Pierwszym napotkanym klopotem byla denerwujaca captcha podczas zakladania konta na stronie githuba, jednakze gdy to sie udalo zadanie wydawalo sie naprawde proste. Strona github jest zbudowana bardzo przejrzyscie i jest latwa w obsludze, przez co tworzenie nowych plikow nie bylo zadnym problemem. Troche czasu zajelo mi takze korzystanie z nowych dla mnie jezykow programowania takich jak csv czy json oraz kodowanie w html, ktore nadal nie jest dla mnie oczywiste. Zadanie jednak okazalo sie niezbyt czasochlonne i bardzo spodobala mi sie praca z githubem.
